Film Characters & MBTI Personality Analysis

A data-driven analysis of MBTI personality types of characters in popular films. Used Python, Selenium for web scraping, and machine learning (XGB Regressor) to analyze how character personality types influence film popularity and audience engagement. Discovered key MBTI pairs (like INFP-INTJ) that correlate with higher film success.

Technologies Used: Python, Jupyter Notebook, Selenium WebDriver, Pandas, NumPy, Matplotlib, Seaborn, NetworkX, scikit-learn, XGBoost, Random Forest, RegEx, BeautifulSoup, Chrome WebDriver, statistical analysis libraries.

Contribution: I developed the web scraping code using Selenium and Python to collect film character data. Created multiple visualizations including network analysis of character relationships, time-series analysis of personality trends across decades, and distribution charts of personality types across film genres. Implemented and ran regression models to identify correlations between character personalities and film success metrics.

Satellite Nighttime Imagery for GDP & University Emissions Prediction

A data science project using satellite nighttime imagery to predict GDP and university emissions. Discovered a clear relationship between GDP and light emissions, which led to work on predicting university emissions. Currently a work in progress.

Technologies Used: Python, Geopandas, scikit-learn, Pandas, NumPy, Matplotlib, Ridge and Lasso regression models, k-fold cross validation, geospatial analysis libraries.

Contribution: Created time-series visualizations of nighttime imagery data, implemented and ran multiple regression models including Ridge and Lasso regression achieving over 95% accuracy. Performed k-fold cross validation to ensure model reliability and mapped universities by longitude and latitude coordinates.

Deepfake Detection

A computer vision project using the Celeb-DF dataset to predict whether an image is a deepfake or authentic. The system extracts frames from videos and applies deep learning models to identify manipulated media.

Technologies Used: Python, TensorFlow/Keras, OpenCV, Transfer Learning, Convolutional Neural Networks, Cross-validation, Flask/Web interface.

Contribution: Built both custom models from scratch and implemented transfer learning approaches. Performed cross-validation to evaluate model performance. Helped develop a user interface where users can upload images for deepfake detection. Currently working on expanding the training dataset and refining the model architecture to improve accuracy.

Network Analysis of Amazon Fine Foods Reviews

A comprehensive analysis of Amazon Fine Foods review data to uncover patterns and relationships between reviewers, products, and review content. The project involved extracting data in Python and conducting advanced network analysis in R.

Technologies Used: Python for data extraction, R for network analysis, igraph, Network visualization libraries, Sentiment analysis tools, Statistical analysis packages.

Contribution: Created network graphs to visualize relationships between reviewers and products. Conducted eigenvector analysis to identify influential nodes. Developed user classification system (novice, intermediate, expert) based on review patterns. Performed sentiment analysis correlating review length with positive/negative sentiment. Identified and analyzed subgroups/communities within the review network. Implemented k-fold analysis to validate findings.
